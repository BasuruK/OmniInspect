## Project Overview

OmniInspect, also presented in the UI as OmniView, is a Go terminal application for Oracle AQ trace inspection, local configuration management, and optional webhook forwarding. It combines a deterministic Bubble Tea v2 TUI with Oracle AQ integration, BoltDB-backed local state, and deployable PL/SQL assets.

## Quick Reference

- **Project type:** CLI application with strong desktop/TUI characteristics
- **Tech stack:** Go, Bubble Tea v2, Lip Gloss v2, Bubbles v2, BoltDB, Oracle AQ, ODPI-C, PL/SQL
- **Entry point:** `cmd/omniview/main.go`
- **Architecture pattern:** Hexagonal core plus adapter-driven infrastructure and event-driven terminal UI

## Getting Started

### Prerequisites

- Go 1.24+
- Oracle Instant Client 23.7+
- C toolchain for CGO
- `make`

### Setup

```bash
make install
```

### Build and Run

```bash
make build
make run
```

### Test

```bash
make test
```
